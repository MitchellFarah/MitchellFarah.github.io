My Final Project!

This game below is called Champ! It is named after the plane that I trained and earned my pilots license in.
The game is a vertical scrolling game with a plane 'The Champ' and pylons that you use W,A,S and D to navigate forward, backward, left and right to avoid the pylons.
The game has four difficulties, easy, medium, hard and extreme. With each step up the program draws more pylons more frequently.
The goal of the game is to survive as long as possible.
